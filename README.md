#  Student Marks System (Flask)

##  Project Description

The Student Marks System is a web-based application developed using Flask (Python) that helps manage and store students' academic marks. It allows users to add student details, calculate total marks, average, and grades automatically, and view all student records in a structured format.



##  Features

* Add student details (Name, ID, Marks)
* Automatic calculation of:

  * Total Marks
  * Average
  * Grade (A, B, C, Fail)
* View all student records
* Simple and user-friendly interface



## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* Database: SQLite

##  Installation & Setup

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000/



##  Purpose

The main purpose of this project is to reduce manual work in managing student marks and provide a fast, accurate, and efficient way to handle academic records.
